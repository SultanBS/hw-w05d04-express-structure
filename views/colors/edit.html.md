# Palettes Edit Page

Edit pages render forms to make requests to edit data within our database.

### Method:
- `GET` and `PUT`
### Action:
- `palette/:palette_id/colors/edit`
### Inputs:
- `Two buttons for submit and cancel, two labels, and nav for show color `

## Other things we want to render:
- `redirect to palette/:palette_id/colors/:color_id`
